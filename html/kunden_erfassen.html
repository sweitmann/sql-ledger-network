Kunden erfassen - Run my Accounts Hilfe

WICHTIG:
  • Achten Sie auf den korrekten Mehrwersteuercode. 
  • Wenn Sie dem Kunden Rechnungen mit Einzahlungsschein schicken wollen, setzen Sie ein Häckchen bei "Einzahlungsschein".

Erfassen der Kundendaten

  1. Unter Debitoren - Kunden - Kunde erfassen geben Sie alle Kunden-Stammdaten ein, die Sie benötigen.
  2. Wählen Sie den richtigen Mehrwertsteuercode für den Kunden aus. Wenn Sie dem Kunden Rechnungen mit 7.6 % MWSt. stellen wollen, dann setzen Sie das Häckchen auf "MWSt 7,6%".
  3. Setzen Sie ein weiteres Häckchen auf "MWST im Preis enthalten", sofern Sie dem Kunden Preise inkl. MWSt. offerieren.
  4. Die Zahlungsbedingungen für den Kunden geben Sie in Tagen an.
  5. Sofern Sie dem Kunden mit der Rechnung jeweils einen Einzahlungsschein schicken wollen, setzen Sie bei "Einzahlungsschein" ein Häckchen.
