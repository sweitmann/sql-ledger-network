Leistungen erfassen - Run my Accounts Hilfe

  1. Sie rapportieren Ihre Stunden unter Projekte - Leistung erfassen. 
  2. Wählen Sie Ihren Namen unter "Mitarbeiter" aus. 
  3. Wählen Sie die Projektnummer.
  4. Geben Sie das Datum ein.
  5. Wählen Sie den zu verwendenden Artikel unter Dienstleistungscode aus. Dazu geben Sie die Artikelnummer oder ein Teil der Artikelnummer ein und drücken die Eingabetaste resp. "Aktualisieren".
  6. Unter Leistungsstart erfassen Sie die Startzeit, unter Leistungsstop die Endzeit (es müssen nicht alle Felder ausgefüllt werden).
  7. Unter "Nicht verrechenbar" geben Sie die Anzahl Stunden an, welche Sie Ihrem Kunden nicht verrechnen können.
  8. Mit einem Klick auf "Aktualisieren" wird das Feld Verrechenbar berechnet.
  9. Der Stundensatz kann bei bedarf abgeändert werden.
  10. Unter Bemerkungen erfassen Sie falls gewünscht einen Kommentar.
  11. Der Datensatz wird mit einem Klick auf "Speichern" gesichert.
Erfasste Leistungen modifizieren
  1. Unter Projekte - Berichte - Leistungen grenzen Sie die gesuchte Leistung ein.
  2. Klicken Sie auf die ID und bearbeiten Sie die Leistung wie gewünscht.
  3. Klicken Sie auf "Speichern". Beachten Sie, dass Sie bereits verrechnente Leistungen nicht mehr verändern können..
