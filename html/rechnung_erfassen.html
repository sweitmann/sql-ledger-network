Rechnung erfassen - Run my Accounts Hilfe

Vorab:

  • Erfassen Sie Ihren Kunden unter Debitoren – Kunden – Kunden erfassen, falls dieser noch nicht im Drop-Down Menu erscheint.
  • Erfassen Sie Artikel unter Waren und Dienstleistungen – Dienstleistung erfassen / Ware erfassen, falls diese noch nicht erfasst wurden
WICHTIG:
  • Vergessen Sie nicht, die Rechnung am Schluss des Vorganges zu buchen.

Rechnung erstellen

  1. Erstellen Sie eine Rechnung, indem Sie unter Debitoren - Debitorenrechnungen auswählen.
  2. Wählen Sie den Kunden aus dem Drop-Down Menu aus.
  3. Die Rechnungsnummer wird automatisch generiert. Alternativ geben Sie eine von Ihnen bestimmte Rechnungsnummer ein.
  4. Geben Sie den Titel der Rechnung unter „Beschreibung“ ein.
  5. Wählen Sie den Artikel aus: dazu geben Sie entweder die Artikelnummer ein (oder auch ein Fragment dieser Nummer), falls Ihnen diese bekannt ist; oder geben die Beschreibung (oder auch ein Fragment dieser Beschreibung) ein. Drücken Sie die Enter-Taste oder "Aktualisieren". Danach wählen Sie Ihren Artikel aus der Liste aus.
  6. Ändern Sie die Daten wie Beschreibung, Menge, Einheit und Preis nach Ihrem Bedarf, falls dies nötig ist.
  7. Drücken Sie „Aktualisieren“ (alternativ Alt-U), falls Sie einen weiteren Artikel erfassen wollen. Fahren Sie mit Punkt 3 oben fort.
  8. Nach dem Sie alle Artikel erfasst haben, können Sie eine Bemerkung anbringen, die auf die Rechnung gedruckt werden soll.
  9. Klicken Sie auf „MWST im Preis enthalten“, falls die MWST im Preis enthalten ist.

Vorauszahlungen erfassen

  1. Falls der Kunde eine Barzahlung oder eine Vorauszahlung leistet, können Sie dies bereits auf der Rechnung berücksichtigen und entsprechend ausweisen.
  2. Unter Zahlungen fügen Sie das Zahldatum ein.
  3. Unter Betrag fügen Sie den Zahlungsbetrag ein.
  4. Wählen Sie das Konto aus, auf welches der Kunde bezahlt hat.

Positionen sortieren:

Falls Sie die Positionen innerhalb der Rechnung sortieren wollen haben Sie dazu folgende Möglichkeiten:
  1. Sortieren Sie die Positionen, in dem Sie die Reihenfolge mittels Vergabe von Nummer unter "Pos." verändern. TIPP: Wenn Sie z.B. ein Artikel an die 2. Stelle bewegen wollen, ohne alle Positionen umzuschreiben, geben Sie im Feld "Pos." 1.5 ein und drücken Sie den Knopf "Aktualisieren". Die Reihenfolge ändert sich wie gewünscht.
  2. Sie haben die Möglichkeit, die Positionen nach Warengruppen zu sortieren. Klicken Sie dazu ganz unten unter "Gruppieren nach ->" auf das Feld "Warengruppen". Ihre Positionen werden auf der Rechnung nach Warengruppen sortiert. Zusätzlich erscheint der Titel der Warengruppe sowie eine Zwischensumme pro Warengruppe.

Rechnung drucken:

  1. Wählen Sie im Drop-Down Menu ganz unten „Rechnung“
  2. Wählen Sie im Drop-Down Menu das gewünschte Ausgabeformat
  3. Wählen Sie im Drop-Down Menu „Bildschirm“
  4. Klicken Sie auf den Knopf „Drucken“
  5. Es wird für Sie eine PDF Datei generiert, die sie ausdrucken oder speichern können.

Rechnung über Run my Accounts drucken und versenden (bei Aktivierung der entsprechenden Option):

  1. Wählen Sie im Drop-Down Menu ganz unten „Rechnung“
  2. Wählen Sie im Drop-Down Menu das gewünschte Ausgabeformat
  3. Zur Kontrolle der Rechnung wählen Sie im Drop-Down Menu „Bildschirm“, klicken auf den Knopf „Drucken“
  4. Wählen Sie im Drop-Down Menu „Warteschlange“
  5. Kontrollieren Sie unter Massenverarbeitung - Warteschlange - Debitorenrechnung ob die Rechnung auch tatsächlich in die Warteschlange gedruckt wurde. 
  6. Nachdem Run my Accounts für Sie die Rechnung ausgedruckt und versandt hat, ist die Warteschlange leer. So haben Sie stets die Kontrolle, welche Rechnungen versandt wurden.

Einzahlungsschein drucken:

  1. Wählen Sie im Drop-Down Menu ganz unten „Einzahlungsschein“
  2. Wählen Sie im Drop-Down Menu das gewünschte Ausgabeformat
  3. Wählen Sie im Drop-Down Menu „Bildschirm“
  4. Klicken Sie auf den Knopf „Drucken“
  5. Es wird für Sie eine PDF Datei generiert, die sie ausdrucken oder speichern können.
  6. Drucken Sie den Einzahlungsschein auf einem Formular mit orangem Einzahlungsschein aus.

Rechnung Buchen

  1. Nachdem alles korrekt erfasst und die Rechnung ausgedruckt ist, drücken Sie auf den Knopf "Buchen"

  2. Die Rechnung wird verbucht und Sie gelangen erneut auf die Eingabemaske für Rechnungen.
TIPP: Speichern können Sie Ihre Rechnungen nur über den Knopf "Buchen". Bei Bedarf können Sie die gespeicherten Rechnungen später wieder löschen oder editieren (siehe Rechnungen editieren / Rechnung löschen).

Bemerkungen:

  • Die Rechnungsnummer wird automatisch generiert.
  • Falls das Rechnungsdatum vom Tag der Eingabe abweicht, kann es manuell gesetzt werden.
  • Das Fälligkeitsdatum wird aus den Kundenstammdaten errechnet. Es kann manuell verändert werden.
